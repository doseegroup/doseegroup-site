Instagramで飲食店の投稿を見た外国人観光客が最初に確認するのが、**プロフィール**です。

どれだけ投稿の写真が良くても、プロフィールが日本語だけ・情報が不足していると、来店につながりません。逆に、プロフィールが整っているだけで「ここは外国人を歓迎してくれる店だ」という安心感を与えられます。

この記事では、インバウンド集客に効くInstagramプロフィールの設計方法を解説します。

## プロフィールが重要な理由

Instagramのアルゴリズムは、プロフィールへのアクセス数・フォロー率を「このアカウントが価値あるか」の判断材料の一つとして使っています。

つまり、プロフィールが整っているほど：

- 投稿を見た外国人がフォローしてくれる確率が上がる
- 来店前の確認場所として機能する
- 予約・問い合わせのアクションが増える

プロフィールは「集客の入口」です。投稿と同じくらい重要に設計する必要があります。

## ユーザーネームとアカウント名の設定

**ユーザーネーム（@以降）**はURLにも使われるため、検索されやすい名前を選ぶことが重要です。

- できるだけシンプルで覚えやすいもの
- 店名をそのままローマ字表記にするのが基本
- 例：`@sakura_sushi_tokyo`、`@ramen_ichiban_shibuya`

**アカウント名（表示名）**は検索結果に表示されます。

- 店名に加えてジャンル・エリアを入れると検索されやすくなる
- 例：`桜鮨 | Tokyo Sushi`、`一番ラーメン | Shibuya Ramen`
- 日本語と英語を組み合わせることで国内・海外両方に対応できる

## プロフィール文（bio）の書き方

プロフィール文は150文字以内に収める必要があります。**日本語と英語を両方入れる**のが基本です。

**含めるべき情報：**
1. ジャンル・コンセプト（何の店か）
2. 営業時間
3. 定休日
4. 予約方法（電話・ウェブ予約のいずれか）
5. 最寄り駅またはエリア

**プロフィール文の例：**

```
🍣 Authentic Edo-style sushi in Ginza
📍 2 min walk from Ginza Station
🕐 Lunch 11:30-14:00 / Dinner 17:00-22:00
📅 Closed Tuesdays
📩 Reservations: Link below

---

銀座の本格江戸前鮨｜銀座駅徒歩2分
ランチ11:30〜 / ディナー17:00〜 / 火曜定休
ご予約はリンクよりどうぞ
```

絵文字を使うと視覚的に整理しやすくなります。ただし使いすぎると読みにくくなるため、項目の区切りに使う程度が適切です。

## ハイライトの設計

ハイライトはプロフィールに常時表示されるストーリーズのまとめです。外国人ゲストが「来店前に知りたいこと」を整理する場所として最適です。

**設置すべきハイライトの項目：**

| ハイライト名 | 内容 |
|---|---|
| Menu / メニュー | 主要メニューの写真・価格 |
| Access / アクセス | 地図・最寄り駅からの道順 |
| Hours / 営業時間 | 曜日別の営業時間・定休日 |
| Reservation / 予約 | 予約方法・注意事項 |
| FAQ | よくある質問（英語対応・アレルギー等） |

**ハイライトのカバー画像**は統一感を持たせると、プロフィールの印象が大きく上がります。シンプルなアイコンや文字のみのデザインで十分です。

ハイライトのタイトルも**英語か日英併記**にするのがポイントです。「メニュー」だけだと英語話者には伝わりません。

## リンクの活用

プロフィールのリンクは「来店アクション」につながる最重要の導線です。

**リンクに設定すべきもの：**
- 予約ページ（一番優先度が高い）
- Googleマップのリンク
- ウェブサイトのメニューページ

複数のリンクを設定したい場合は、**Linktr.ee などのリンクまとめツール**を活用すると、1つのリンクから複数のページに案内できます。

## よくある失敗パターン

**失敗1：日本語のみのプロフィール**
外国人ゲストにとって、日本語だけのプロフィールは「自分向けではない」というサインに映ります。

**失敗2：情報が少なすぎる**
「おいしいご飯を提供しています🍽️」のような内容では、外国人ゲストが来店判断に必要な情報を得られません。

**失敗3：ハイライトが未整備または古い**
ハイライトが「1年前のキャンペーン」だけになっているケースがよくあります。定期的に見直し、常に最新の情報を置くことが重要です。

**失敗4：リンクが設定されていない**
プロフィールにリンクがない場合、来店意欲があってもアクションを起こす場所がなくなります。

## まとめ

インバウンドに強いInstagramプロフィールの要点：

1. **ユーザーネーム・アカウント名**：英語またはローマ字表記で検索されやすく
2. **プロフィール文**：日英併記で営業時間・アクセス・予約方法を記載
3. **ハイライト**：Menu / Access / Hours / Reservation を英語タイトルで整備
4. **リンク**：予約ページまたはリンクまとめページを設定

プロフィールは一度作れば資産になります。最初に丁寧に設計しておくことで、その後の投稿効果が変わります。
